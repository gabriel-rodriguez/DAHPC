# DAHPC 2026

This repository contains the documentation and examples for the second lesson of the DAHPC class in the Master in HPC at the Universidade da Coru√±a.

### Instructions to set up a working environment to run the examples:

1. Install Miniforge for your OS: 

```
$ https://conda-forge.org/download/
```

2. Create your environment using Mamba:

```
$ mamba env create -f dahpc-environment.yml
```

3. Activate the DAHPC environment:

```
$ mamba activate dahpc-2026
```

It is possible that you need to activate your shell before you can activate the environment. If so, follow the on-screen instructions.

4. Launch JupyterLab:

```
$ jupyter-lab
```
